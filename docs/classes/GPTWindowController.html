Class Description

GPTWindowController is a subclass of NSWindowController that controls an AquaTerm window. Every instance of a GPTWindowController holds a reference to the rendering AQTView. GPTWindowController assists in setting the model for the view by assuring that the window nib is loaded before the model received from the AQTBuilder object is handed to the view.
